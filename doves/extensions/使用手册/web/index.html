白鸽系统

分布式智能体任务执行框架 — 通过 CLI 发起任务，由自主运行的 Doves 智能体拉取并执行，全链路 Noise NX 加密。

快速开始 CLI 命令参考 扩展开发

核心特性

全链路加密

所有组件间通信统一 Noise NX 加密 TCP，无明文 HTTP，单端口 3003。

自主智能体

Doves 自主轮询拉取任务、调用 LLM 执行、多实例并发，支持扩展能力插件。

自然语言交互

通过 CLI 或 IM 通道（微信/钉钉/飞书）用自然语言下达任务。

扩展生态

16+ 官方扩展，涵盖编码、文档、视频处理、数据分析等能力，支持自定义扩展开发。

分布式任务队列

MongoDB 原子锁任务获取，支持依赖阻塞、检查点恢复、自动重试。

多模型支持

百炼、DeepSeek、智谱 GLM 官方接入，支持自定义 Claude/GPT/Gemini 代理。

系统架构

CLI ══加密TCP══▶ Server (:3003) ◀══加密TCP══ Doves
 │                    │                         ▲
 │                    ▼                         │
 │            MongoDB / OSS / 向量记忆           │
 │            (仅 Server 有权直接访问)           │
 │                                              │
 └══════加密TCP（同账号直连：管理/扩展）═════════┘

三条通信线路全部为 Noise NX 加密 TCP。Server 是纯数据网关（不调用 LLM），Doves 是自主执行引擎（拉取任务、调用 LLM），CLI 是用户入口。
