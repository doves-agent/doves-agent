鸽子管理

鸽子（Dove）是白鸽系统的智能 Agent 实例。通过鸽子管理命令可以注册、配置、查看和管理 Agent 的生命周期及能力。

dove dove

Agent 注册与管理命令，控制鸽子的注册、配置、密钥和通道权限。

dove dove [action] [options]

Actions

Action Description
register 注册新鸽子
list / ls 列出所有鸽子
info <id> 查看鸽子详情
delete / rm <id> 删除鸽子
key <action> 密钥管理（generate/show/rotate）
config <key> [value] 查看/修改鸽子配置
use <id> 切换当前使用的鸽子
current 显示当前鸽子
mcp <action> MCP 服务器管理
capability / cap 能力管理（快捷方式）
channel-permission / cp 通道权限管理

选项

Flag Description Default
-n, --name <name> 鸽子名称 -
-t, --type <type> 鸽子类型 general
-c, --capabilities <list> 能力列表（逗号分隔） -
--id <id> 指定鸽子 ID -
--force 强制执行（跳过确认） false

MCP 选项

Flag Description Default
--mcp-add <name> 添加 MCP 服务器 -
--mcp-remove <name> 移除 MCP 服务器 -
--mcp-url <url> MCP 服务器地址 -
--mcp-cmd <command> MCP 启动命令 -

通道权限选项

Flag Description Default
--channel <name> 通道名称（wechat, cli, api） -
--grant 授予权限 -
--revoke 撤销权限 -

示例

# 注册新鸽子
dove dove register -n "数据分析师" -t analyst -c "数据分析,Python,可视化"

# 列出所有鸽子
dove dove ls

# 查看鸽子详情
dove dove info dove_abc123

# 切换当前鸽子
dove dove use dove_abc123

# 查看当前鸽子
dove dove current

# 生成新密钥
dove dove key generate --id dove_abc123

# 修改鸽子配置
dove dove config max_concurrency 5

# 添加 MCP 服务器
dove dove mcp add --mcp-add weather --mcp-url "http://localhost:8080"

# 设置通道权限
dove dove cp --channel wechat --grant --id dove_abc123

# 删除鸽子
dove dove rm dove_abc123 --force

dove capability

能力管理命令（别名：cap）。查看和刷新鸽子的能力声明，用于任务分配匹配。

dove capability [action] [options]

Actions

Action Description
refresh 刷新能力声明（重新检测）
list / ls 列出当前能力
info <name> 查看能力详情
report 生成能力报告

选项

Flag Description Default
-d, --dove <id> 指定鸽子 ID 当前鸽子
-a, --all 显示所有鸽子的能力 false
--json JSON 格式输出 false

示例

# 刷新当前鸽子能力
dove cap refresh

# 列出所有鸽子的能力
dove cap ls -a

# 查看指定鸽子的能力
dove cap ls -d dove_abc123

# 查看特定能力详情
dove cap info "视频处理"

# 生成能力报告
dove cap report --json

dove skill

技能管理命令（别名：skills）。启用或禁用鸽子的技能模块。

dove skill [action] [target] [options]

Actions

Action Description
list / ls 列出可用技能
enable <name> 启用技能
disable <name> 禁用技能
status <name> 查看技能状态

选项

Flag Description Default
-c, --category <cat> 按分类过滤 全部
-a, --all 包含远程技能市场 false
--json JSON 格式输出 false
--remote 仅显示远程技能 false

示例

# 列出已安装技能
dove skill ls

# 列出所有技能（含远程市场）
dove skill ls -a

# 按分类查看
dove skill ls -c "数据处理"

# 启用技能
dove skill enable "网页抓取"

# 禁用技能
dove skill disable "网页抓取"

# 查看技能状态
dove skill status "视频处理"

提示：技能启用后，鸽子的能力声明会自动更新，无需手动刷新。

相关命令

  • dove task - 任务会根据鸽子能力自动分配
  • dove config - 鸽子全局配置
  • dove app - 扩展应用（提供额外技能）
