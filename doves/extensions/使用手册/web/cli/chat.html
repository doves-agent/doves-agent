对话命令

对话命令是白鸽 CLI 的核心功能，支持流式对话、会话管理和长连接模式。

dove chat

核心对话命令，支持流式输出、多轮会话、并发分解等高级特性。发送消息给鸽子并实时接收响应。

dove chat [message] [options]

选项

Flag Description Default
-c, --conversation <id> 指定会话 ID，继续已有对话 新建会话
-l, --list 列出最近会话 false
--select 交互式选择会话继续 false
-m, --model <model> 指定使用的模型 配置默认值
-s, --stream 启用流式输出 true
-p, --profile <name> 使用指定人设配置 默认人设
--no-split 禁用任务自动分解 false
--no-parallel 禁用并发执行 false
--no-flash 禁用 Flash 快速模式 false
--max-concurrency <n> 最大并发数 自动
--max-depth <n> 最大递归深度 自动
--execution-mode <mode> 执行模式：decomposition_first | interleaved | pipeline | direct 自动选择
--plan-first 先规划再执行（等同 decomposition_first） false
--tool-level <level> 工具权限等级：safe | caution | dangerous safe
--enable-ability <name> 启用指定能力 -
--disable-ability <name> 禁用指定能力 -
--disable-tool <name> 禁用指定工具 -
--gateway 使用网关路由 false
--async 异步模式，提交后立即返回 false
--ws 使用 WebSocket 通信 false
--direct 直连鸽子（同账号） false
--no-direct 强制走 Server 中转 false
--debug 显示调试信息 false

执行模式说明

  • decomposition_first - 先将任务完整分解为子任务，再依次执行
  • interleaved - 分解与执行交替进行，边分析边执行
  • pipeline - 流水线模式，前一步输出作为后一步输入
  • direct - 直接执行，不做任何分解

示例

# 简单对话
dove chat "今天天气怎么样"

# 继续已有会话
dove chat -c abc123 "接着上面的话题"

# 使用指定模型并启用计划优先模式
dove chat -m claude-opus-4-20250514 --plan-first "帮我分析这个项目的架构"

# 异步提交复杂任务
dove chat --async "整理并翻译这份文档"

# 直连鸽子，绕过 Server
dove chat --direct "查看当前鸽子状态"

# 限制并发和深度
dove chat --max-concurrency 3 --max-depth 2 "并行处理这三个文件"

# 使用危险工具等级（需确认）
dove chat --tool-level dangerous "清理临时文件"

dove conv

会话管理命令，查看和管理历史对话记录。

dove conv [action] [options]

Actions

Action Description
list / ls 列出会话（默认 action）
info <id> 查看指定会话详情

选项

Flag Description Default
-l, --limit <n> 列出条数限制 20
--json JSON 格式输出 false

示例

# 列出最近会话
dove conv list

# 查看指定会话详情
dove conv info 6478a3f2e1b

# JSON 格式输出（便于脚本处理）
dove conv ls --json | jq '.[] | .id'

dove session

WebSocket 长连接模式，保持持久连接，支持实时任务推送和交互式命令。适合需要长时间交互的场景。

dove session [options]

选项

Flag Description Default
-g, --gateway 通过网关建立连接 false

Session 内部命令

命令 说明
/help 显示帮助信息
/task [id] 查看任务状态
/tasks 列出所有活跃任务
/conv [id] 切换/查看会话
/new 开启新会话
/status 显示连接状态
/clear 清屏

示例

# 进入 session 模式
dove session

# 通过网关连接
dove session -g

# session 内操作示例
> 你好，帮我查看今天的任务
> /tasks
> /conv abc123
> /new
> /status

提示：Session 模式下连接断开会自动重连，未完成的任务不会丢失。
