CLI 命令参考

白鸽 CLI 基于 Commander.js 构建，二进制名称为 dove，提供 30+ 命令覆盖对话、任务、鸽子管理、存储、服务运维等全部功能。

两种使用模式

命令行模式：直接执行命令，如 dove chat "你好"，适合脚本和快速操作。

交互模式：直接运行 dove 进入交互式 Shell，支持命令补全和历史记录。

# 命令行模式
dove chat "帮我总结今天的工作"

# 交互模式
dove
> chat 帮我总结今天的工作
> task list
> exit

全局选项

Flag Description Default
--wait 等待异步任务完成后再返回 false
--debug 启用调试输出，显示详细日志 false
--version 显示当前版本号 -

免认证命令

  • dove login / dove logout - 登录登出
  • dove setup - 系统安装
  • dove init - 初始化向导
  • dove ping - 连通性测试
  • dove info - 系统信息
  • dove log - 日志查看
  • dove service - 服务管理
  • dove test - 测试命令
  • dove app create / dove app validate / dove app dev - 扩展开发
