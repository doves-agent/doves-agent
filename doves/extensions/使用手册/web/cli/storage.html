存储命令

存储命令管理白鸽系统的文件、Git 仓储、OSS 临时空间和向量记忆。

dove file

文件操作命令，支持读写、上传下载等基本文件操作。

dove file [action] [path] [options]

Actions

Action Description
read <path> 读取文件内容
write <path> 写入文件
list [dir] 列出目录文件
delete <path> 删除文件
upload <path> 上传本地文件到 OSS
download <path> 从 OSS 下载文件

选项

Flag Description Default
-o, --output <path> 输出路径（下载时使用） 当前目录
-c, --content <text> 写入内容（write 时使用） stdin

示例

# 读取文件
dove file read /data/report.md

# 写入文件
dove file write /data/note.txt -c "这是一段备注"

# 列出目录
dove file list /data/

# 上传文件到 OSS
dove file upload ./local-file.pdf

# 下载文件
dove file download oss://reports/2024-q1.pdf -o ./downloads/

# 删除文件
dove file delete /data/temp.txt

dove storage

高级存储管理命令，涵盖 Git 仓储、OSS 临时空间和目录权限管理。

dove storage [action] [args...] [options]

Git 仓储 Actions

Action Description
status 查看 Git 存储状态
mount <repo> 挂载 Git 仓库
clone <url> 克隆远程仓库
quota 查看存储配额
quota-set <size> 设置存储配额
quota-delete 删除配额限制

OSS 临时空间 Actions

Action Description
temp-create 创建临时空间
temp-urls 获取临时空间访问 URL
temp-list 列出临时空间
temp-upload <file> 上传到临时空间
temp-copy <src> <dst> 临时空间内复制
temp-finalize <id> 归档临时空间为永久存储
temp-delete <id> 删除临时空间

目录管理 Actions

Action Description
dir-list 列出托管目录
dir-create <name> 创建托管目录
dir-info <name> 查看目录信息
perm-grant 授予目录访问权限
perm-revoke 撤销目录访问权限
perm-list 列出目录权限

文件操作 Actions

Action Description
file-read <path> 读取存储中的文件
file-write <path> 写入文件到存储
file-delete <path> 删除存储中的文件
file-list [dir] 列出存储目录内容

选项

Flag Description Default
-p, --path <path> 目标路径 -
-t, --target <target> 目标位置 -
-s, --source <source> 源位置 -
--owner <uid> 所有者 UID 当前用户
--task <id> 关联任务 ID -
--hash 计算文件哈希 false
--capacity <size> 容量限制（如 1GB） -
--inodes <n> 最大文件数 -
--preserve 保留原始属性 false
--sync-to <path> 同步到本地路径 -
-f, --file <file> 指定文件 -

示例

# 查看存储状态
dove storage status

# 挂载仓库
dove storage mount my-project

# 克隆远程仓库
dove storage clone https://github.com/user/repo.git

# 查看配额
dove storage quota

# 创建临时空间
dove storage temp-create

# 上传到临时空间
dove storage temp-upload ./output.zip

# 归档为永久存储
dove storage temp-finalize tmp_abc123

# 创建托管目录
dove storage dir-create "项目文件" --capacity 5GB --inodes 1000

# 授予目录权限
dove storage perm-grant -p "/项目文件" --owner user_456

# 读取存储文件
dove storage file-read /projects/readme.md

# 列出存储目录
dove storage file-list /projects/

dove memory

向量记忆管理命令（别名：mem）。管理鸽子的长期记忆，支持语义搜索。

dove memory [action] [query] [options]

Actions

Action Description
search / s 语义搜索记忆
list / ls 列出记忆条目
stats 记忆统计信息
add / a 手动添加记忆
delete / del / d 删除记忆条目

选项

Flag Description Default
-u, --user <uid> 指定用户 当前用户
-t, --type <type> 记忆类型：event_trigger, conversation, knowledge, instruction 全部
-p, --page <n> 页码 1
-l, --limit <n> 每页条数 20
--threshold <n> 相似度阈值（0-1） 0.7
--top-k <n> 返回最相关的 K 条 5
--json JSON 格式输出 false

记忆类型说明

  • event_trigger - 事件触发器（当某事发生时自动响应）
  • conversation - 对话记忆（重要对话片段）
  • knowledge - 知识（用户教授的事实信息）
  • instruction - 指令（用户的偏好和要求）

示例

# 语义搜索
dove memory search "上次讨论的项目方案"

# 按类型搜索
dove mem s "每周报告" -t instruction

# 调整相似度和数量
dove mem s "服务器部署" --threshold 0.8 --top-k 10

# 列出所有记忆
dove mem ls

# 按类型列出
dove mem ls -t knowledge

# 查看统计
dove mem stats

# 手动添加知识记忆
dove mem add "用户偏好使用 Claude 模型处理复杂任务" -t instruction

# 删除记忆
dove mem del mem_abc123

# JSON 输出（便于脚本处理）
dove mem ls --json | jq '.[] | select(.type == "knowledge")'

注意：记忆一旦删除无法恢复。建议删除前先用 dove mem ls 确认记忆内容。
