服务命令

服务命令管理白鸽系统的后台进程、Web 面板、运行日志和用量统计。

dove service

服务生命周期管理命令，控制 Server 和 Doves 进程的启停与开机自启。

dove service [action] [options]

Actions

Action Description
start 启动服务
stop 停止服务
restart 重启服务
status 查看服务状态
logs 查看服务日志
startup 管理开机自启
check 检查服务健康
save 保存当前进程列表
web 启动 Web 管理面板

选项

Flag Description Default
-t, --type <type> 服务类型：server 或 dove 全部
-p, --prod 生产模式启动 false
--lines <n> 日志行数 50
--json JSON 格式输出 false
-a, --all 所有服务 false

Startup 子选项

Flag Description Default
--enable 启用开机自启 -
--disable 禁用开机自启 -
--status 查看开机自启状态 -

示例

# 启动所有服务
dove service start

# 仅启动 Server
dove service start -t server

# 生产模式启动
dove service start -p

# 查看状态
dove service status

# JSON 格式状态（便于监控脚本）
dove service status --json

# 重启鸽子服务
dove service restart -t dove

# 停止所有服务
dove service stop -a

# 查看服务日志
dove service logs -t server --lines 100

# 检查健康
dove service check

# 启用开机自启
dove service startup --enable

# 查看自启状态
dove service startup --status

# 保存当前进程列表
dove service save

dove web

启动 Web 管理面板，提供图形化界面管理白鸽系统。

dove web [options]

选项

Flag Description Default
-p, --port <port> 监听端口 5173
--host <host> 绑定地址 127.0.0.1
--no-open 不自动打开浏览器 false
--dev 开发模式（热更新） false

示例

# 默认启动 Web 面板
dove web

# 指定端口且不自动打开浏览器
dove web -p 8080 --no-open

# 允许局域网访问
dove web --host 0.0.0.0

# 开发模式
dove web --dev

Web 面板提供对话管理、任务监控、鸽子状态、配置编辑等图形化功能，适合不熟悉命令行的用户。

dove log

日志查看命令，实时跟踪各模块的运行日志。

dove log [module] [options]

Modules

Module Description
server Server 端日志
doves 鸽子运行日志
llm LLM 调用日志

选项

Flag Description Default
-n, --lines <n> 显示行数 50
-f, --follow 实时跟踪（类似 tail -f） false
-e, --error 仅显示错误 false
--json JSON 格式输出 false

子命令：clean

dove log clean [--days <n>]

清理过期日志文件。

示例

# 查看 Server 日志
dove log server

# 实时跟踪鸽子日志
dove log doves -f

# 仅查看 LLM 错误日志
dove log llm -e

# 查看最近 200 行
dove log server -n 200

# 清理 7 天前的日志
dove log clean --days 7

# JSON 格式（便于日志分析工具）
dove log server --json -n 100

dove stats

Token 用量统计面板，查看各维度的 LLM 调用消耗。

dove stats [options]

选项

Flag Description Default
--by-model 按模型分组统计 false
--by-provider 按提供商分组统计 false
--by-day 按天分组统计 false
--today 仅统计今天 false
--week 最近一周 false
--month 最近一月 false
--from <date> 起始日期（YYYY-MM-DD） -
--to <date> 结束日期（YYYY-MM-DD） -
--all 所有时间 false

示例

# 查看今日用量
dove stats --today

# 按模型统计本周
dove stats --week --by-model

# 按提供商统计本月
dove stats --month --by-provider

# 按天查看趋势
dove stats --by-day --from 2026-05-01 --to 2026-05-21

# 全量统计
dove stats --all --by-model --by-provider

提示：dove stats 是免认证命令，即使登录过期也可以查看本地缓存的统计数据。
