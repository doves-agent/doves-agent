配置命令

配置命令管理白鸽系统的网关、密钥、模型和人设等核心设置。

dove config

通用配置管理命令，涵盖网关设置、API 密钥管理、模型配置等。

dove config [action] [key] [value] [options]

Actions

Action Description
list 列出所有配置项
get <key> 获取配置值
set <key> <value> 设置配置值
gateway 管理当前网关
gateways 列出所有网关
reset 重置配置为默认值
set-key <provider> <key> 设置 API 密钥
get-key <provider> 查看 API 密钥（脱敏）
list-keys 列出所有已配置密钥
test-key <provider> 测试密钥有效性
del-key <provider> 删除 API 密钥
intent-model 设置意图识别模型
reasoning-model 设置推理模型
planning-model 设置规划模型
vision-model 设置视觉模型
flash-model 设置快速模型
model-settings 查看模型设置
model-defaults 重置模型为默认配置
chat-log 查看对话日志配置
chat 对话相关配置

选项

Flag Description Default
-p, --provider <name> 指定提供商（openai, anthropic, deepseek 等） -
-m, --model <name> 指定模型名称 -
-i, --index <n> 网关索引 0
-r, --role <role> 模型角色 -
-k, --key <key> API 密钥值 -
-t, --test 设置后自动测试 false

示例

# 列出所有配置
dove config list

# 设置配置项
dove config set server.url "tcp://192.168.1.100:3003"

# 获取配置
dove config get server.url

# 管理 API 密钥
dove config set-key anthropic sk-ant-xxx
dove config set-key openai sk-xxx
dove config list-keys
dove config test-key anthropic

# 设置推理模型
dove config reasoning-model -p anthropic -m claude-opus-4-20250514

# 设置快速模型
dove config flash-model -p anthropic -m claude-sonnet-4-20250514

# 查看模型设置
dove config model-settings

# 网关管理
dove config gateway
dove config gateways

# 重置为默认
dove config reset

dove model

模型管理命令，刷新可用模型列表和组织模型配置。

dove model [action] [options]

Actions

Action Description
refresh 从提供商刷新可用模型列表
organize 整理模型配置（去重、排序）
list 列出已配置模型

选项

Flag Description Default
-p, --provider <name> 按提供商过滤 全部
-m, --model <name> 指定模型 -
-o, --output <format> 输出格式（table/json） table
--dry-run 预览变更，不实际执行 false

示例

# 刷新所有提供商的模型
dove model refresh

# 仅刷新 Anthropic 的模型
dove model refresh -p anthropic

# 列出所有模型
dove model list

# JSON 格式列出
dove model list -o json

# 整理模型配置
dove model organize

# 预览整理结果
dove model organize --dry-run

dove profile

人设配置管理，定义鸽子在对话中的角色、性格和行为模式。

dove profile [action] [name] [options]

Actions

Action Description
list 列出所有人设
show <name> 显示人设详情
create <name> 创建新人设
delete <name> 删除人设
tags 列出所有标签

选项

Flag Description Default
-t, --tag <tag> 按标签过滤 -
-k, --keyword <kw> 按关键词搜索 -

示例

# 列出所有人设
dove profile list

# 按标签过滤
dove profile list -t "专业"

# 显示人设详情
dove profile show "技术助手"

# 创建新人设
dove profile create "翻译官"

# 删除人设
dove profile delete "旧人设"

# 列出所有标签
dove profile tags

人设与对话：在 dove chat 中通过 -p 选项指定人设，鸽子会按照人设定义的性格和规则进行对话。

相关命令

  • dove chat -p - 使用指定人设对话
  • dove dove config - 鸽子级别配置
  • dove status - 查看当前配置状态
