快速开始

本指南帮助你在 5 分钟内完成白鸽系统的安装与首次对话。

前置条件

依赖 要求 说明
Node.js >= 18 推荐 20 LTS
MongoDB >= 6.0 Docker 部署可自动拉起
LLM API Key 至少一个 支持百炼 / DeepSeek / 智谱GLM

安装

提供两种部署方式，Docker 为推荐方式。

方式一：Docker 部署（推荐）

一行命令完成安装，自动拉起 Server、Doves 和 MongoDB：

curl -fsSL https://get.dove.ai/install.sh | bash
./install.sh --docker --with-mongo

提示：如果已有外部 MongoDB 实例，可使用 --skip-mongo 跳过内置数据库部署。

方式二：裸机部署

git clone https://github.com/anthropic/doves.git
cd 白鸽系统
npm run 安装依赖

首次配置

运行初始化向导，只需 3 步即可完成基础配置：

dove init

向导将引导你完成以下配置：

  1. Server 地址 — 填写服务端连接地址（本地默认 http://localhost:3003）
  2. 账号信息 — 创建管理员账号或登录已有账号
  3. LLM 密钥 — 配置至少一个模型提供商的 API Key

启动服务

通过 PM2 一键启动 Server 和 Doves：

dove service start

服务启动后，Server 监听 3003 端口，Doves 自动轮询任务队列。

首次对话

一切就绪，发送你的第一条消息：

dove chat "你好，帮我写一首关于春天的诗"

Doves 将通过 LLM 生成回复并输出到终端。

验证系统状态

使用以下命令检查各组件健康状态：

dove status

正常输出示例：

┌──────────┬────────┬─────────┐
│ 组件     │ 状态   │ 版本    │
├──────────┼────────┼─────────┤
│ Server   │ 运行中 │ 1.2.0   │
│ Doves    │ 运行中 │ 1.2.0   │
│ MongoDB  │ 已连接 │ 7.0.4   │
│ LLM      │ 可用   │ -       │
└──────────┴────────┴─────────┘

交互模式：直接运行 dove（不带参数）即可进入交互式对话模式，支持多轮连续对话、上下文记忆和历史回溯。
